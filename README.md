# github-client
Android-приложение для просмотра репозиториев GitHub с использованием Jetpack Compose, библиотек okhttp, retrofit2 и koin.

Функциональные возможности


- Авторизация через Personal Access Token
- Просмотр списка репозиториев пользователя (первые 10)
- Детальная информация о репозитории:
  - Статистика (stars, forks, watchers)
  - Ссылка на web-страницу
  - README файл

Технологический стек
- Язык - Kotlin 
- UI - Jetpack Compose + Material 3 
- Архитектура - MVVM + Clean Architecture 
- Сеть - Retrofit 2 + OkHttp 3 
- DI - Koin 
- Асинхронность - Kotlin Coroutines + Flow 
- Навигация - Jetpack Navigation Compose 
- Хранилище - SharedPreferences 
- Минимальная версия - Android API 26 (Android 8.0)
